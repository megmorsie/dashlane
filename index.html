Why Use Dashlane?

Security Benefits

  • Secure Sharing of Login Credentials/Notes
  • Encourages Use of Strong Passwords
  • Discourages Duplicate Passwords
  • Alerts You When a Company Has a Security Breach
  • As opposed to Excel Sheets, Knowledge Base
  • Security Score, Password Generator
  • Example: Agent Portal and personal banking
  • So you can update your password. Example: Lynda.com

Convenience/Workflow Benefits

  • Auto-Login (which can be turned off)
  • Changing a Password Without Needing to Tell Everyone That Has it
  • No Need to Memorize All Passwords
  • Can Use App to Get Into Systems on Any Device
  • Will show where to turn auto-login off later.
  • Auto-Syncs
  • Less time resetting passwords
  • Shared passwords on P drive won’t be accessible on personal devices
